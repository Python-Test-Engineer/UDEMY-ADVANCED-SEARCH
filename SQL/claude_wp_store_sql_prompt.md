# Create products, customers and orders

```
For WordPress
Create and SQL script that creates the following tables:
wp_store_products - base loosely on woocommerce product talbe definitons
wp_store_customers
wp_store_orders
Create insert statement for 40 sample products, 5 customers and 50 orders
Create one final SQL script
```

It took a few rounds to remove errors.

## PRO TIP:

Start with just a few of each to get it working and also avoid wasting free context tokens.

Once working, ask for a larger set.
